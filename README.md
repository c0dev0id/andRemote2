# Android DMD Remote 2 emulator

This app emulates the DMD Remote 2 as an on screen overlay.

This is not possible with stock android capabilities!

Therefore the App requires [Shizuku](https://shizuku.rikka.app), to send keycodes to the foreground application.
Please install and set up Shizuku first: [Setup Shizuku via Wireless Debugging](https://shizuku.rikka.app/guide/setup/#start-via-wireless-debugging)
